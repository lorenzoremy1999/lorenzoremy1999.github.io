# Lorenzo Remy — Portfolio Site

Static one-page portfolio + hosted PDF portfolios. No build step, no dependencies.

```
index.html   ← the website
pdf/         ← your two portfolio PDFs (linked + downloadable)
img/         ← images pulled from the PDFs
README.md
```

## Deploy on GitHub Pages (≈10 min, free)

1. Create a new **public** repo. For the cleanest URL, name it
   `lorenzoremy1999.github.io` (gives you `https://lorenzoremy1999.github.io`).
   Any repo name works too — you'll just get `…github.io/repo-name`.
2. Upload **everything in this folder** (drag-and-drop into the repo's
   "Add file → Upload files", or `git push`). Keep the `pdf/` and `img/`
   folders intact.
3. Repo **Settings → Pages → Build and deployment**:
   set **Source = Deploy from a branch**, **Branch = main / (root)**, Save.
4. Wait ~1 minute, then open the URL Pages shows you. Done.

(search for `TODO LORENZO`): email, LinkedIn, GitHub, phone.
